# 《Java并发编程实战》中文笔记


这本书不愧是豆瓣评分9.0的神书，它很系统的讲述了JUC的各种知识，由浅入深，先讲怎么用再讲原理。就是原理部分讲的不是很多，可以继续看《Java并发编程的艺术》作为补充。

这个笔记主要用于我自己复习JUC，相当于对原书的一个摘要。

> 原书地址：https://book.douban.com/subject/10484692/
>
